# 🌟 AIFriends
AIFriends 是一款基于大语言模型（LLM）的虚拟陪伴平台，无需编写代码，即可快速创建、个性化定制并分享专属虚拟伴侣、知心好友等虚拟角色。平台集成前沿语音处理技术与 RAG 架构，致力于打破虚拟与现实的边界，带来自然、沉浸式的情感陪伴与智能交互体验。

## ✨ 核心特性
* **🎭 深度角色定制与社区分享**
  * 支持无限制创建个性化虚拟角色
  * 全维度自定义配置：性格特质、背景故事、对话风格、专属音色一键设定
  * 开放社区生态：原创角色可一键分享，实现角色共创与传播

* **🎙️ 沉浸式实时语音交互**
  * **实时语音通话**：告别纯文本聊天，支持与虚拟角色实时语音连线互动
  * **智能语音识别（ASR）**：高精度识别语音内容，精准理解用户意图
  * **语音合成（TTS）& 音色克隆**：支持文字转语音，可克隆绑定专属音色，还原真人听觉体验

* **🧠 硬核 AI 能力驱动**
  * **大模型中枢**：基于 LangChain 框架，实现提示词工程优化与长效记忆管理
  * **函数调用（Function Calling）**：赋予虚拟角色调用外部工具能力，支持实时信息查询、任务执行等功能
  * **私有知识库（RAG）**：挂载自定义文档与角色设定，让虚拟角色拥有专属知识体系与连贯上下文记忆

## 🛠️ 技术栈
项目采用前后端分离架构，技术选型如下：

### 前端 (Frontend)
* **核心框架**: Vue 3 + Vite
* **样式方案**: Tailwind CSS
* **状态管理/路由**: Pinia + Vue Router

### 后端 (Backend)
* **核心框架**: Django
* **API 开发**: Django REST Framework (DRF)
* **数据库**: MySQL / PostgreSQL

### AI & 算法层 (AI & LLM)
* **开发框架**: LangChain
* **语音技术**: OpenAI Whisper / ChatTTS / VITS
* **向量数据库**: Milvus / Chroma
